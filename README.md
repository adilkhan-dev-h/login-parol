---

##  Общая идея

Этот код реализует простую систему:

* регистрации пользователей
* входа в систему

Все данные хранятся во временной памяти (в словаре), поэтому после закрытия программы они исчезают.

---

## Хранение пользователей

```python
users = {}
```

Создаётся пустой словарь:

* ключ — это логин (`username`)
* значение — это пароль (`password`)

Пример после регистрации:

```python
users = {
    "admin": "1234",
    "user1": "qwerty"
}
```

---

##  Функция регистрации

```python
def register():
```

Что она делает:

1. Просит пользователя ввести логин:

   ```python
   username = input("Введите логин: ")
   ```
2. Проверяет, существует ли уже такой логин:

   ```python
   if username in users:
   ```

   * если да → выводит сообщение и завершает функцию
3. Если логин новый:

   * запрашивает пароль
   * сохраняет логин и пароль в словарь

   ```python
   users[username] = password
   ```

 Итог: добавляет нового пользователя.

---

## Функция входа

```python
def login():
```

Что происходит:

1. Пользователь вводит логин и пароль

2. Проверка:

   ```python
   if username in users and users[username] == password:
   ```

   * существует ли логин
   * совпадает ли пароль

3. Если всё верно → вход выполнен

4. Если нет → ошибка

 Итог: проверяет, правильные ли данные ввёл пользователь.

##  Основной цикл программы

```python
while True:
```

Это бесконечный цикл, который:

* показывает меню
* ждёт выбор пользователя

```python
print("1 - Регистрация")
print("2 - Вход")
print("3 - Выход")
```

Дальше:

```python
choice = input("Выберите действие: ")
```

И проверка:

* `"1"` → вызывается `register()`
* `"2"` → вызывается `login()`
* `"3"` → программа завершается (`break`)
* другое значение → ошибка

---

##  Как всё работает вместе

1. Программа запускается
2. Пользователь выбирает действие
3. Может:

   * зарегистрироваться (добавить себя в словарь)
   * войти (проверка данных)
4. Программа работает до выхода

---

## Ограничения

Этот код учебный, потому что:

* данные не сохраняются (после закрытия всё пропадает)
* пароли хранятся как обычный текст
* нет защиты (любой может подобрать пароль)

---

##  Итог

Это базовая модель системы авторизации:

* регистрация → добавление данных
* вход → проверка данных
* цикл → управление программой
